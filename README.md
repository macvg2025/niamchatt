# NiamChat

A real-time chatting room application with public and private rooms, message reactions, and custom themes.

## Features
- Public chat room for everyone
- Private rooms with shareable 6-character codes
- Message reactions (like, dislike, emoji reactions)
- Image upload support
- 10 custom themes with visual effects
- Admin system with special privileges
- First-time user experience with big buttons

## Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/niamchat.git
cd niamchat
# Backend
cd backend
npm install

# Frontend (in a new terminal)
cd ../frontend
npm install
# Terminal 1: Start backend server
cd backend
npm start

# Terminal 2: Start frontend development server
cd frontend
npm run dev
